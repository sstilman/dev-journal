## 技术栈

本项目使用 [Docusaurus](https://docusaurus.io/) 构建。

## 安装

```bash
yarn
```

## 本地开发

```bash
yarn start
```

启动本地开发服务器，默认访问 http://localhost:3000

## 构建

```bash
yarn build
```

生成静态文件到 `build` 目录

## 部署

```bash
yarn deploy
```

使用 GitHub Pages 部署
