# Linux 基础

本系列教程帮助嵌入式开发者掌握 Linux 系统基础知识。

## 定位

面向嵌入式开发者的 Linux 入门教程，聚焦实际开发中最常用的知识点。

**前置要求**：了解基本的计算机操作即可

## 学习路径

```
Linux 基础
├── 入门篇
│   ├── Linux 系统启动流程      # 从按下电源到登录界面
│   ├── 常用命令速查            # 开发中真正会用到的命令
│   └── 文件系统结构            # Linux 文件系统目录布局
├── 进阶篇
│   ├── Shell 脚本基础          # 自动化运维入门
│   ├── 用户与权限              # Linux 权限模型
│   └── 进程与服务              # systemd 的使用
└── 嵌入式专题
    ├── 交叉编译环境            # 在 PC 上编译 ARM 程序
    ├── NFS 与 TFTP             # 远程调试准备
    └── Yocto 快速入门          # 构建嵌入式 Linux
```

2. **实践中遇到问题** — 查技术随笔找解决方案
3. **做项目记录** — 把过程写入项目札记
